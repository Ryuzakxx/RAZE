## `> OVERVIEW`

RAZE ├и un assistente AI con interfaccia terminale sci-fi, pensato per girare **completamente offline**.  
Supporta sia modalit├а testuale che vocale, con riconoscimento vocale locale via Faster-Whisper e sintesi TTS.

---

## `> FEATURES`

| | |
|---|---|
| ЁЯОЩя╕П **Voice Mode** | STT con Faster-Whisper (`small`), TTS locale, waveform in tempo reale |
| ЁЯТм **Text Mode** | Chat con LLM locale, storico conversazione, temi personalizzabili |
| ЁЯОи **Temi** | Dark / Light / Cyber тАФ cambio live senza restart |
| ЁЯУб **Offline** | Nessuna API esterna richiesta |

## `> SETUP`

```bash
git clone https://github.com/Ryuzakxx/RAZE.git
cd RAZE
pip install -r requirements.txt
python main.py
```

> Assicurati di avere i modelli TTS nella cartella `models/` e i video in `assets/`.
